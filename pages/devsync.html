    DevSync

    Husky Coding Project Group, UI/UX Lead, Frontend

    As part of the Husky Coding Project, I was matched with a team to help build DevSync, a swipe-based platform that connects college developers with projects and collaborators. It streamlines the process of discovering and joining teams or finding co-founders for side projects.

    Project Summary:

    • Spearheaded UI/UX design for DevSync using Figma, creating an intuitive swipe-based interface to connect college developers with peers and projects.
    • Collaborated on frontend development using React, contributing to key pages and components in a full-stack application.
    • Worked on a cross-functional team using Node.js, MySQL, and cloud hosting to build a verified, student-only platform for project matchmaking.
    • Presented DevSync at the Husky Coding Project Showcase, highlighting accessibility, intentionality, and student engagement.

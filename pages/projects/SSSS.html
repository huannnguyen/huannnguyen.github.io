    SmartSecure Security System

    Embedded Systems RTOS + Project

    Implemented a real-time preemptive operating system (FreeRTOS) to coordinate multiple hardware and software tasks in a time-sensitive environment. Developed a security box system integrating LEDs, a servo, an ultrasonic sensor, LCD display, keypad, potentiometers, and a speaker to demonstrate practical RTOS applications.

    Demo

    The system displays real-time status on the LCD, responds to keypad input, triggers alarms, plays melodies, and controls the servo—all coordinated through FreeRTOS tasks.

    Our Approach

    • 5 FreeRTOS tasks for LEDs, sensors, servo, keypad, and speaker.
    • Location- and input-based alerts using an ultrasonic sensor and keypad.
    • 3D-printed casing for a functional security box.
    • In-app logic for alarms, pin verification, and interactive feedback through LEDs and display.

    Process

    • User research & device exploration: Studied Arduino libraries and external components.
    • Prototyping: Task implementation in FreeRTOS with semaphores and timers.
    • Testing & Iteration: Integrated multiple devices and optimized task scheduling.

    Key Learnings

    • Semaphore and task management is crucial to avoid conflicts in RTOS.
    • Priority and timing handling enable smooth concurrent operations.
    • Combining hardware and software requires careful planning and testing.

    Impact

    Successfully created a working, multi-task embedded system with hardware-software integration. Learned practical FreeRTOS applications, task coordination, and the challenges of real-time embedded programming.

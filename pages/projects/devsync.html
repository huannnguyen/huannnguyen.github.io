    DevSync

    Platform Connecting College Developers & Projects

    DevSync is a swipe-based platform that connects college students to projects, teams, and collaborators. I served as UI/UX lead and implemented the frontend.

    üîó Link to DevSync's GitHub repo


    The Problem

    Finding teammates as a student is hard‚Äîespecially if you're new to tech spaces or not connected to a campus network. Existing platforms often focus on resumes rather than meaningful matches.

    Our Solution

    We built a platform where:

    • Students swipe on projects or peers based on interests
    • Profiles include bio, skills, and resume
    • Mutual matches connect students to collaborators
    • Verified student access ensures a high-trust environment

    Tech Stack

    • Frontend: React
    • Backend: Node.js + MySQL
    • Hosting: Cloud server

    Key Learnings

    • Building a swipe-based UI for meaningful matches
    • Designing inclusive interfaces for college students
    • Implementing frontend functionality efficiently with React

    Impact

    DevSync makes it effortless for students to discover projects and collaborators, lowering barriers to entry in tech spaces and fostering motivated peer connections.
